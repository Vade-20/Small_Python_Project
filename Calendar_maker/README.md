# Calendar Maker

This Python script generates and prints a calendar for a given year and month. It utilizes the `datetime` and `calendar` modules to calculate and display the calendar.

## Functionality

The script provides the following functionality:

- Prompting the user to enter a year and month for which the calendar will be generated.
- Generating a calendar for the specified year and month.
- Printing the calendar to the console in a nicely formatted manner.
